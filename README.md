# DevOps Engineer Portfolio

A modern, responsive portfolio website for DevOps Engineers featuring:

## Features

- **Dark/Light Mode Toggle** - Seamless theme switching with localStorage persistence
- **Smooth Animations** - Scroll-triggered animations and floating elements
- **Responsive Design** - Mobile-friendly layout
- **Modern UI/UX** - Glassmorphism, gradients, and contemporary design patterns
- **Interactive Elements** - Hover effects, glitch animations, typing effects

## Sections

1. **Hero** - Eye-catching introduction with animated tech icons
2. **About** - Professional summary with key statistics
3. **Skills** - Technical expertise showcase (Docker, K8s, AWS, Terraform, etc.)
4. **Projects** - Featured DevOps projects with tags
5. **Contact** - Contact form and social links

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript
- Font Awesome Icons

## Customization

### Update Personal Information

Edit `index.html`:
- Line 35: Change the main title
- Line 36: Update subtitle
- Lines 60-62: Modify statistics
- Lines 145-165: Update contact information
- Lines 145-165: Add your email, LinkedIn, GitHub links

### Modify Projects

Edit the project cards in `index.html` (lines 115-145):
- Change project titles, descriptions, and tags
- Add/remove projects as needed

### Adjust Colors

Edit CSS variables in `style.css` (lines 8-16):
```css
--primary: #00d4ff;
--secondary: #7c3aed;
```

## Deployment

### Local Testing
Simply open `index.html` in a browser.

### Deploy to GitHub Pages
1. Create a GitHub repository
2. Push these files
3. Enable GitHub Pages in repository settings

### Deploy to Netlify/Vercel
1. Drag and drop the folder to Netlify/Vercel
2. Site will be live instantly
